# tdl-gui
基于tdl下载工具的GUI交互窗口，交互式下载和上传
# 使用方法

- 将 `tdl.exe` 和当前 GUI 程序放在同一个目录下。
- 登录操作、环境设置、代理设置等仍通过原有的 CMD 指令执行。
- 登录成功后，即可使用 GUI 进行下载或上传操作。
